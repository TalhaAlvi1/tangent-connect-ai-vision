## 🛠️ Troubleshooting

### If Training Fails:
- Check that ultralytics is installed: `pip install ultralytics`
- Ensure all 48 images were copied correctly
- Verify label files were created during labeling

### If Detection Isn't Accurate:
- Lower confidence threshold in settings.py (CONFIDENCE_THRESHOLD)
- Add more training images (50-100 images per class recommended)
- Retrain with more epochs (100-150)

### If Model Doesn't Load:
- Check that training completed successfully
- Verify the model file exists at the expected path
- Ensure you're selecting the "best.pt" file, not "last.pt"
